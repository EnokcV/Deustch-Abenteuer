# Mapas y referencias visuales

Esta carpeta contiene imágenes de referencia para el mapa de Alemania y los interiores de las ciudades.

## Archivos

- `germany-map-reference.png` — Mapa completo de Alemania con las 4 ciudades destacadas (Berlin, Frankfurt, München, Hamburg) y los 6 interiores (Brandenburger Tor, Hafenmeister Büro, Bank, Biergarten, Warenlager, Rathaus)

## Estilo artístico

Las imágenes son pixel art de alta calidad. Se usan como referencia visual para:

1. El `WorldMapScene` que renderiza el mapa procedural de Alemania
2. Las escenas interiores que recrean el ambiente de cada ciudad
3. Los markers de ciudad con su estilo distintivo

Si en el futuro quieres imágenes reales de fondo (parallax), colócalas aquí en formato 320×240 (un tile-screen) o 320×180 (HD).
